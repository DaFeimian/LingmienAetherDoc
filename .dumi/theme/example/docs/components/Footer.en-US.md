## API

| Name    | Type                 | Description                                  |
| ------- | -------------------- | -------------------------------------------- |
| columns | `RcProps['columns']` | Column configuration                         |
| bottom  | `RcProps['bottom']`  | Bottom content                               |
| theme   | `RcProps['theme']`   | Theme, optional values are `dark` or `light` |
